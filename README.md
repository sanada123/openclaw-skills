## Skills

### 🏃 [lifestyle-optimizer](./lifestyle-optimizer/)

**The operating system for high-performance founders.**

Not wellness fluff — tactical protocols that work inside a 60-hour week with a newborn.

| Module | What You Get |
|--------|-------------|
| 🧠 Productivity | 5-block day, GTD for CTOs, deep work protocol, shutdown ritual |
| ⚡ Energy | Sleep science, minimum-dose exercise, decision-free nutrition |
| 👶 Parenting | Sleep guides 0-12m, milestones 0-36m, working parent protocols |
| 🚨 Emergency | "I'm overwhelmed", "baby won't sleep", burnout warning system |

**For:** Founders, CTOs, startup engineers, anyone building a company while raising kids.

#### Install

**OpenClaw / Claude Code:**
```bash
# Clone to your skills directory
git clone https://github.com/sanada123/openclaw-skills.git /tmp/oc-skills
cp -r /tmp/oc-skills/lifestyle-optimizer ~/.openclaw/workspace/skills/
# Or for Claude Code:
# cp -r /tmp/oc-skills/lifestyle-optimizer ~/.claude/skills/
```

**Verify:**
```bash
openclaw skills info lifestyle-optimizer
# Should show: ✓ Ready
```
