## Quick Install

**OpenClaw:**
```bash
git clone https://github.com/sanada123/openclaw-skills.git /tmp/oc-skills
cp -r /tmp/oc-skills/{skill-name} ~/.openclaw/workspace/skills/
rm -rf /tmp/oc-skills
```

**Claude Code:**
```bash
git clone https://github.com/sanada123/openclaw-skills.git /tmp/oc-skills
```

**Codex CLI:**
```bash
git clone https://github.com/sanada123/openclaw-skills.git /tmp/oc-skills
```

## Why These Skills Exist

**Arabic:** 400 million speakers, zero dedicated AI content skills. Every "Arabic" AI output reads like Google Translate because no one taught the model which dialect to use where.

**Hebrew:** 9 million speakers, zero dedicated skills. AI Hebrew sounds like a government form from 1995. Israelis deserve content that sounds like them.

**Lifestyle:** Every productivity skill is for 9-5 workers. Every parenting skill ignores that you might also be shipping code at midnight. We built for both.
