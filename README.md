## Skills

### 🏃 [lifestyle-optimizer](./lifestyle-optimizer/)

**The operating system for high-performance founders.**

Not wellness fluff — tactical protocols that work inside a 60-hour week with a newborn. Evidence-based, research-cited, and immediately actionable.

| Module | What You Get |
|--------|-------------|
| 🧠 **Productivity** | 5-block day, GTD for leaders, deep work protocol, shutdown ritual |
| ⚡ **Energy** | Sleep science (cited), minimum-dose exercise, decision-free nutrition |
| 👶 **Parenting** | Baby sleep guide 0-12m, milestones 0-36m (WHO/AAP), working parent protocols |
| 🚨 **Emergency** | "I'm overwhelmed" protocol, "baby won't sleep" protocol, burnout warning system |
| 📚 **References** | Deep work, sleep science, month-by-month milestones, burnout recovery (4 files) |

**For:** Founders · CTOs · Freelancers · Engineers · New parents · Anyone building a company while raising kids.

**Sources:** Cal Newport, Andrew Huberman, AAP Sleep Guidelines, WHO/CDC Developmental Milestones, Maslach Burnout Inventory, BJ Fogg habit research.

## Why This Exists

Most productivity skills are written for people with predictable 9-5 schedules. Most parenting advice assumes you're not also shipping code at midnight. Most AI agent skills focus on coding — nobody made one for the person *behind* the keyboard.

This repo fills that gap. If you're building a startup, managing a team, and trying to be present for your kid's first steps — these protocols are for you.
